# Cas 02 — La recherche Outlook ne fonctionne plus

## Symptômes
- Recherche vide
- Résultats incomplets
- Indexation bloquée

## Analyse
Problème souvent dû à :
- Windows Search corrompu
- Indexation incomplète
- OST/PST endommagé

## Diagnostic
1. Vérifier l’état de l’indexation :
   Paramètres → Recherche → Windows Search

2. Vérifier l’OST :
   Emplacement :
%LOCALAPPDATA%\Microsoft\Outlook

3. Réindexation manuelle :
Panneau de configuration → Options d’indexation

## Solution appliquée
- Reconstruction de l’index
- Réparation Outlook (Quick Repair)
- Vérification/renouvellement du fichier OST

## Vérification finale
- Résultats de recherche complets et rapides
