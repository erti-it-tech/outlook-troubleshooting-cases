# Cas 10 — Erreur Send/Receive (0x800CCC...)

## Symptômes
- Impossible d’envoyer ou recevoir
- Erreur Send/Receive

## Analyse
Problèmes fréquents :
- connectivité
- certificat
- fichier OST

## Diagnostic
1. Test connectivité :
Test-NetConnection outlook.office365.com -Port 443

2. Réparation compte Outlook

3. Test hors VPN

## Solution appliquée
- Réparation M365
- Recréation profil si nécessaire

## Vérification finale
- Messages envoyés/reçus avec succès
