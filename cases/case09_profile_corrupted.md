# Cas 09 — Profil Outlook corrompu

## Symptômes
- Outlook démarre mal
- Messages d’erreur “profil endommagé”

## Diagnostic
1. Panneau de configuration → Mail  
2. Créer un nouveau profil  
3. Le définir par défaut

## Solution appliquée
- Création d’un nouveau profil propre

## Vérification finale
- Outlook fonctionne normalement
